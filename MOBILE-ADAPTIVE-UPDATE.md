# Mobile adaptive update

Updated for phones and tablets:
- collapsible mobile sidebar with backdrop and close button;
- full-width mobile search form;
- responsive lead list and detail panel;
- horizontally swipeable CRM columns;
- responsive login, invite, recovery, and set-password screens;
- mobile-safe form controls (16px to prevent iOS zoom);
- responsive admin area and tables;
- removed global minimum page width and horizontal page overflow.

After replacing files, run:

```bash
npm install
npm run build
```
