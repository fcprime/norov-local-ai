# Norov Local AI v2.2

## Що змінило

- Пошук тепер працює за логікою **ваша послуга → кому продаємо**.
- Приклад: «Хімчистка меблів» → «Готелі».
- Geoapify шукає саме цільовий тип бізнесу, а не конкурентів, які надають таку саму послугу.
- Lead Score та всі штучні оцінки повністю прибрані.
- Результати сортуються за наявністю відкритих контактів.
- Офер і ймовірні потреби формуються на основі вашої послуги та вибраного сегмента.

## Запуск

1. Створіть `.env` у корені проєкту.
2. Додайте:

```env
API_PORT=8787
GEOAPIFY_API_KEY=ваш_ключ
```

3. Запустіть:

```bash
npm install
npm run dev
```

## Приклад пошуку

- Ваша послуга: `Хімчистка меблів`
- Кому продаємо: `Готелі`
- Країна: `Poland`
- Місто: `Wrocław`
- Радіус: `25 км`
