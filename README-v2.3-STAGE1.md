# Norov Local AI v2.3 — Stage 1

## Що змінено
- 27 країн ЄС + Велика Британія, Норвегія, Швейцарія та Ісландія.
- Поле «Що ви продаєте?» приймає текст будь-якою мовою. Воно використовується для персоналізації офера, а пошук компаній виконується за вибраним цільовим сегментом.
- Розширено список цільових сегментів.
- Geoapify-категорії замінено на підтримувані широкі категорії.
- Знайдені компанії більше не потрапляють у CRM автоматично. У картці компанії є кнопка «Додати в CRM».

## Запуск
1. Скопіюйте `.env.example` у `.env`.
2. Вставте `GEOAPIFY_API_KEY`.
3. Виконайте `npm install`.
4. Виконайте `npm run dev`.
