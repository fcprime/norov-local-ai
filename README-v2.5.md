# Norov Local AI v2.5 — Sales Engine

- Порожній стартовий екран: старі компанії не показуються до першого пошуку.
- Універсальні тексти для будь-якої послуги та сегмента.
- Канали: Email, WhatsApp, Facebook, Instagram, LinkedIn, SMS, Call Script, Follow-up 1/2.
- Тексти можна редагувати вручну.
- Швидке переписування: коротше, ввічливіше, більше довіри, менш продажно.
- Чернетки зберігаються локально в браузері.
- Кнопка «Почати роботу» додає ліда в CRM.
- Звернення можна зберегти в історію взаємодій CRM.

Ця версія працює без платного AI API: генератор локальний та шаблонно-контекстний. Підключення справжньої мовної моделі можна додати пізніше окремим модулем.
